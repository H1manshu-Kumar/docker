## 🐳 Docker Commands Used (Proof of Work)

### ✅ Building Image
```bash
docker build -t python-app:latest .
```

### ✅ Running Container
```bash
docker run -p 5000:5000 python-app:latest
```

### ✅ Listing Running Containers
```bash
docker ps
```
